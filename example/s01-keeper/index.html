这是 S01《守站》 的一套满配参考，给你看"开工前先备好 Reference"具体长什么样。四类参考各司其职：角色锁长相、场景锁空间、风格锁调性、镜头锁构图。守站全程是同一台机器——这正是 S01 的考点。A fully-prepared reference set for S01 "The Keeper", showing what "collect references before you start" actually looks like. Four kinds, each with a job: Character locks the look, Location the space, Style the tone, Shot the framing. The Keeper stays one consistent machine throughout — exactly what S01 tests.

C风格/情绪参考 · Style · MoodStyle · Mood

锁定色彩、质感、调性。一句话：冷钢蓝打底，一抹活的暖琥珀是希望，去饱和冷灰是希望之死。Locks color, texture, tone. In one line: cold steel-blue base, one warm living amber for hope, desaturated grey for its death.
